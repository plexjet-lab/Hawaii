## 📖 프로젝트 소개

하와이의 아름다움과 매력을 소개하는 **풀 페이지 랜딩 사이트**입니다.  
순수 HTML과 CSS만으로 제작되었으며, 빅 아일랜드를 중심으로 하와이의 자연·문화·액티비티를 풍부하게 담았습니다.

---

## ✨ 주요 기능

| 섹션 | 내용 |
|------|------|
| 🏝️ **Hero** | 전체 화면 배경 영상과 알로하 인트로 |
| 🌟 **Highlights** | 하와이를 선택해야 하는 4가지 이유 |
| 🖼️ **Gallery** | 하와이 대표 풍경 포토 그리드 |
| 🗺️ **Islands** | 오아후 · 마우이 · 카우아이 · 빅 아일랜드 소개 |
| 🌋 **Big Island** | 빅 아일랜드 전용 심층 섹션 (명소 6곳 · 액티비티 6종 · 갤러리 · 여행 팁) |
| 📊 **Stats** | 하와이 핵심 수치 인포그래픽 |
| 🤿 **Activities** | 서핑, 화산 하이킹, 루아우 파티 |
| 📩 **CTA** | 여행 시작 유도 섹션 |

## 🌋 빅 아일랜드 심층 소개

이 프로젝트의 핵심 콘텐츠는 **빅 아일랜드(하와이 섬)** 전용 섹션입니다.

### 반드시 가야 할 6곳
- 🔥 **킬라우에아 화산** — 지구상에서 가장 오래 활동 중인 화산 (UNESCO 세계유산)
- ⭐ **마우나케아 정상** — 해발 4,207m, 세계 최고 수준의 별하늘
- 🏖️ **푸날루우 검은 모래 해변** — 화산 용암이 만든 새까만 해변, 하와이 거북 서식지
- 🦈 **만타레이 야간 다이빙** — 세계 10대 다이빙 스팟
- 🌿 **와이피오 계곡** — 하와이 왕족의 신성한 비밀 계곡
- ☕ **코나 커피 농장** — 세계 3대 커피 산지 직접 체험

### 빅 아일랜드 액티비티
용암 보트 투어 · 마우나케아 별빛 투어 · 용암 동굴 탐험 · 야생 돌고래 수영 · 아카카 폭포 하이킹 · 파커 목장 승마

## 🛠️ 기술 스택

```
📦 순수 HTML5 + CSS3 (외부 라이브러리 없음)
├── CSS Custom Properties (변수)
├── CSS Grid & Flexbox 레이아웃
├── CSS Transitions & Animations
├── position: sticky 네비게이션
├── backdrop-filter blur 효과
└── 완전 반응형 (모바일 · 태블릿 · 데스크톱)
```

---

## 📱 반응형 지원

| 화면 크기 | 레이아웃 |
|-----------|----------|
| 📱 모바일 (`< 580px`) | 1열 단일 컬럼 |
| 📟 태블릿 (`< 900px`) | 2열 그리드 |
| 🖥️ 데스크톱 (`≥ 900px`) | 3~4열 그리드 |

## 🚀 로컬 실행 방법

별도 설치 없이 바로 실행할 수 있습니다.

```bash
# 저장소 클론
git clone https://github.com/plexjet-lab/Hawaii.git

# 폴더 진입
cd Hawaii

# index.html을 브라우저에서 열기 (더블클릭 또는 아래 명령어)
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```
